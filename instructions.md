A unique, modern, scroll-based "cool" resume should be a cinematic, interactive journey. To create this, your prompt must blend standard portfolio elements with a cohesive visual narrative. The key is defining *how* the elements transition and how they interact with each other.

Here is a prompt designed for generating a highly unique, modern, scroll-based digital resume. It is written to be versatile, suitable for a designer, developer, or modern professional.

---

### Modern Scroll-Based Resume Prompt: The "Interactive Atlas" Theme

> **Project Goal:** Design a highly stylized, futuristic, and unique single-page scroll-based resume. The visual narrative is an "Interactive Atlas," where the user’s background is a dynamic map, and scrolling navigates through interconnected, distinct data environments. The interface must be minimal, using fluid motion rather than traditional borders or boxes.

---

### Design System and Esthetics

* **Primary Esthetic:** Cinematic, minimalist-futurism (e.g., *Blade Runner* meets *Tron*). Use complex, organic shapes (nodes, geometric constellations) rather than rectangles.
* **Color Palette:** Use a dark mode "Cyber-Electric" palette: deep charcoal/navy background (`#080A10`), neon-cyber-cyan (`#00FFFF`), radioactive lime (`#32CD32`), and subtle magenta (`#FF00FF`) accents for highlights and active states.
* **Typography:** Monospaced fonts for technical data (like code snippets or skills), paired with a bold, ultra-wide sans-serif for section headers. Body text should be clean and light-weight.
* **Motion (The Scroll):** As the user scrolls, elements should not just slide up. They should *emerge*: fade in, rotate slightly, or assembly piece-by-piece. The previous section should disassemble or dissolve into the next. Use parallex effect: background elements move slower than the foreground text.

---

### Section breakdown

#### 1. Header (The Entrance)

* **Visual:** A swirling, translucent, 3D biometric fingerprint icon. When the cursor passes, it activates and displays the name and title in glowing text: `USER_NAME // CREATIVE TECHNOLOGIST`.
* **Interaction:** A minimal sidebar nav appears as five subtle dots. A glowing progress bar follows the scroll depth on the far right.

#### 2. About (The Origin)

* **Visual:** The fingerprint expands, dissolving into a network of glowing nodes. This section does not use a photo.
* **Layout:** A two-line, large-scale manifesto-style quote that fills the screen (e.g., `DECODING IMPOSSIBILITY` / `INTO ELEGANCE`). The detailed about text is split into two small, floating columns of text next to the node graph.
* **Content:** Keep it concise and impact-driven.

#### 3. Skills (The Arsenal)

* **Visual:** Transition from nodes to a 'holographic weapons rack.' Each skill category is represented by a unique, rotating 3D abstract geometric shape (e.g., a polyhedron for frontend, a spinning DNA strand for backend, a fractured sphere for design).
* **Interaction:** *Hover* on a shape to activate it. The shape explodes into a tag cloud of specific tools (e.g., React, TypeScript, GraphQL) that orbit the center shape.

#### 4. Education (The Foundation)

* **Visual:** Instead of a traditional timeline, this is a stylized geological strata map or data layers.
* **Layout:** Degrees are represented as illuminated, stacked horizontally, thick horizontal bars. The bottom layer is the foundational degree.
* **Interaction:** Clicking a degree layer highlights key courses or achievements by injecting glowing data streams into the layer, labeled with relevant years (e.g., '2019-2023 | B.S. in Applied Physics').

#### 5. Projects (The Manifest)

* **Visual:** Transition from strata to a high-speed 'tunnel of experiences.' This is a horizontal scroll element *within* the vertical scroll flow.
* **Layout:** Project cards are deep-space-style anomalies—glowing nebulae or black holes. Each project card contains a minimalist logo, a short description, and links as glowing, abstract icons (e.g., GitHub logo, live site icon).
* **Interaction:** Dragging horizontally reveals more projects. Clicking a project temporarily slows the background particle speed.

#### 6. Contact (The Link)

* **Visual:** The atlas terminates into a single, massive, translucent, abstract portal (like a black hole or gateway).
* **Layout:** Links are presented as binary code streams flowing *away* from the portal. Clicking Email, LinkedIn, or GitHub opens a small, dedicated, minimalist terminal window that prints the URL.
* **Footer:** A small, persistent command-line prompt at the bottom: `USER@ATLAS:~$ Waiting for input...`

---

### Unique Twist: The "Personal AI Interface" Overlay

To make it *most unique*, add a functional overlay called "AI.LOGIC."

* **Feature:** A persistent, minimal AI-style prompt/dialogue box remains in the corner of the entire resume (e.g., "Ask me something about this project or skill").
* **Functionality:** If the user type questions (e.g., "What tools were used on Project X?"), the AI interface highlights the relevant data elsewhere on the current screen, effectively turning the resume into a localized search tool of your professional data.
